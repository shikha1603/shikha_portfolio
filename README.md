# Shikha | AI/ML & Developer Portfolio

A minimalist, typography-driven personal portfolio designed to showcase my journey as an AI/ML Engineer and Developer. Built with a focus on clean aesthetics, subtle micro-interactions, and a strictly monochrome palette with beautiful typography.

![Portfolio Preview](Screenshot%20.png) <!-- Update with an actual screenshot if preferred -->

## ✨ Features
- **Typography-Focused Design**: Leveraging `Cabinet Grotesk`, `Instrument Serif`, and `JetBrains Mono` to create strong visual hierarchy.
- **Custom Cursor**: A slick, inverted-color custom cursor that dynamically scales when hovering over interactive elements (projects, flip-cards, tech icons).
- **Responsive Layout**: Fluidly scales from ultra-wide desktops down to mobile screens seamlessly.
- **3D Interactive Flip-Cards**: An elegant grid of milestones and certificates that flip in 3D to reveal images when hovered.
- **Dynamic Tool Arsenal**: Monochromatic logos that burst to full color upon hover, instantly revealing the tool's name.

## 🛠️ Built With
- **HTML5** & **CSS3** (Pure Vanilla CSS, no frameworks used to keep it lightweight)
- **Vanilla JavaScript** (For scroll reveals and the custom cursor logic)
- **Google Fonts** & **Fontshare**
- Icons provided by **SimpleIcons** and **Devicon**

## 💻 Running Locally
There are no complex build steps or dependencies. Just clone this repository and open the file directly in your browser:

```bash
git clone https://github.com/shikha1603/portfolio.git
cd portfolio
```
Then, double-click `index.html` or open it with your favorite Live Server extension in VS Code.

## 📁 Repository Structure
```text
📦 portfolio
```
